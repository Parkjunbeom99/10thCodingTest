
# 재귀 함수 
정의 단계에서 자기 자신을 참조하는 것 
매개변수가 달라질 뿐 함수는 같다 
큰 문제를 부분문제로 만들때 사용
// f(N)  ->
       f(N-1)
            f(N-2)
                f(N-3)
                    f(N-4)
                        f(N-5)
                        
                       
주의사항 
1. 종료조건이 있어야한다. 
2. 종료조건에 도달할 수 있도록 매개변수를 변경해야한다. 
3. 종료조건에 도달하지 못하면 무한루프에 빠진다. 
4. 종료조건에 도달하지 못하면 스택오버플로우가 발생한다. 
5. 반복문으로 될 거 같으면 반복문으로 하자  -> 코스트가 더 많이 든다 

점화식을 만들어서 감을 잡는게 좋다  
점화식은 “다음 값을 이전 값들로부터 계산하는 규칙”이다.
즉, 수열의 생성 규칙을 나타내는 수학식.
