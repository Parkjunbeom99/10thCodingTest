이분 탐색
이분 탐색 (binary search)는 정렬된 배열에서 탐색 범위를 절반씩 줄여나가는 방식으로 특정 값을 찾는 알고리즘이다 
시간 복잡도는 O(logN)이다. -> 이게 탐색 범위가 계속 절반씩 줄어들기 때문이다

이분탐색의 동작 원리 
1. 초기 설정 : 배열의 가장 왼쪽 인덱스를 left, 가장 오른쪽 인덱스를 right로 설정합니다. 
2. 중간 인덱스 계산 : left와 right의 중간 인덱스 mid를 계산합니다. 
3. 값 비교 : mid 인덱스의 값과 찾고자 하는 값(target)을 비교합니다. 
        만약 target과 같으면 mid값 반환 
        작다면 범위를 왼쪽으로 좁힘 
        크다면 범위를 오른쪽으로 좁힘 
4. 반복 : 2~3 과정을 반복

이분 탐색이 병합 정렬과 같이 분할 정복 기법을 사용한다 

C++에서는 내장된 라이브러리 lower_bound 또는 upper_bound를 통해 구현할 수 있다
하지만 이전 작업에서 반드시 정렬을 시켜둬야한다

Ex
vector<int> arr = {1, 3, 6, 9, 10, 21, 22, 30};
int target = 6;
int result = binarySearch(arr, target);


이분탐색을 이용하면 최적화 문제를 결정 문제로 변환가능

최적화 문제: 어떤 값의 최대값이나 최소값을 찾는 문제.
 예를 들어, "가장 적은 질투심을 갖게 형광펜을 나누는 방법을 찾아라"는 최적화 문제.
● 결정 문제: 어떤 조건을 만족하는지 여부를 확인하는 문제입니다
 (Yes or No) 예를 들어, "주어진 질투심 값으로 형광펜을 나눌 수 있는가?"라는 것은 결정 문제


최대증가부분수열 (LIS)
주어진 배열에서 순서를 유지하면서 숫자가 오름차순으로 증가하는 가장 긴 부분 수열을 찾는 문제
