완전 탐색 

완전 탐색은 브루트 포스라고도 불리며 경우의 수를 탐색하는 알고리즘이다 

경우의 수 ?  순열 or 조합이다 

보통 완탐은 (순열 , 조합) + 로직 (DFS, BFS) 로 이루어진다. 시간 복잡도는 1억 이하일 경우 사용

#1 
while이나 For반복문을 이용한 반복문 
 int main() 
 {
    vector<int> v = {1, 3, 2, 5, 6, 7, 8};
    for(int i = 0; i < v.size(); i++){
        if(v[i] == 5)
            { 
               cout << i << "\n"; break; }
            }
           
            int i = 0; 
            
            
           while(i < v.size())
           {
            if(v[i] == 5)
            {
             cout << i << "\n"; break;
             }
             i++;
        }
    return 0; 
 }
 
 #2 재귀함수를 이용한 완전탐색 
 완전탐색으로 되면 무조건 반복문으로 해라 -> 재귀 함수 설명할때도 설명할때도 말함
 
 
반복문으로 구현하기가 힘들다? 또는 어떠한 행위는 반복하는데 매개변수만 수정해서 넘기면 될 거 같다?
 그러면 재귀함수로 하는게 좋다
 
 
백트래킹 

완전탐색 + 가지치기 

완전탐색과의 차이
완전탐색은 가능한 모든 경우의 수를 탐색하여 해를 찾는 방법
하지만 이 과정에서 많은 비효율적인 탐색이 발생할 수 있다.
백트래킹은 이러한 비효율적인 탐색을 줄이기 위해 
"이런 경우의 수는 탐색을 하지 않아도 된다"는 판단을 내리며 불필요한 경로를 배제

가지치기
경로가 목표 해에 도달할 가능성이 없다고 판단되면 더 이상 그 경로를 탐색하지 않고 되돌아가는(Backtracking) 방법

백트래킹은 주로 재귀로 구현이 된다 


완전탐색 - 원복
완전탐색은 가능한 모든 경우의 수를 탐색하여 해를 찾는 방법
하지만 이 과정에서 탐색 후의 상태 값이 다른 경로에 영향을 주면 안된다
그래서 탐색을 한 후 상태를 이전으로 복구하는 과정을 원복이라 한다
